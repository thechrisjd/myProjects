Education

Studying at KEA in IT technology

Currently I'm at university obtaining my degree in web development. The course is split up into three parts: web design, database design and user interface. In web design we are focusing on html, css, php, JavaScript (jQuery and node.js), and python (Django). Through proper web design we seek to build robust full stack web sites. For database design we are creating/administrating JSON databases, MySQL, MariaDB and MongoDB. Finally, interface design focuses on the paper work involed in creating a website. We implement scrum, kanban and other project management techniques to make sure we forfill the customers requirements.

Furthermore, I have an AP degree in IT tech. This of course is made up of three parts programming, networking and server administration. In server administration, we set up multiple virtual networks and implemented server software to maintain the domain, manage users/groups through OU structures, manage the logical topology through sites and manage security/permissions. In most cases we are using a virtual windows environment (2003, 2003 R2, 2008 and 2012), however, in some instances, we used Linux servers. The virtualization software I use is VMware workstation and Virtual box. In the networking part of my course were taught the OSI model (the 7 layers of the internet), the different protocols that reside in each layer, how to perform packet capturing with Cisco packet sniffer/wire shark and how to configure network deceives like routers and switches. The programming languages I learnt were processing, python 3, JavaScript, C and C#. Finally, I also have my comptia network + certificate.
